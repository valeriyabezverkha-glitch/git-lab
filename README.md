Демонстраційний проект для лабораторної роботи з Git.

Цей репозиторій містить просту програму на C++ для обробки даних студентів.

## Встановлення

Склонуйте репозиторій та відкрийте файл main.cpp.

## Використання

Запустіть main.cpp і введіть дані студентів вручну.
